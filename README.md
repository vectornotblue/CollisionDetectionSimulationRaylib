Simple Discrete Collision Detection Simulation with Optimisation Algorithms in C++ using Raylib. Currently only implemented SAP( Sweep and Prune) algorithm.

Red lines represent collision checks that are happening between a pair of balls each frame. 

Collision handling implemented by following Chad Berchek's "2-Dimensional Elastic Collisions without Trigonometry" solution.

With Sweep and Prune (SAP) algorithm collision checks per frame dropped from 4950 to around 300 for 100 balls.
